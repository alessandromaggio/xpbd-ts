# XPBD Physics Engine Product Overview

## Mission
Create a high-performance Extended Position-Based Dynamics (XPBD) physics engine in TypeScript for browser-based applications.

## Core Features
- Extended Position-Based Dynamics simulation
- Browser-optimized performance
- TypeScript type safety
- Modular architecture for easy integration
- Real-time physics simulation

## Target Use Cases
- Interactive web applications
- Game development
- Educational physics simulations
- Creative coding projects
- Scientific visualizations

## Technical Goals
- Zero external dependencies for core engine
- Clean separation between physics engine and rendering
- Hot reload development environment
- Comprehensive example library
